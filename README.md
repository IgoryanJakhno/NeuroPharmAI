# 🧠 Нейро-фарм — ИИ-агент для поиска лекарств и их аналогов

[![GitHub release](https://img.shields.io/github/v/release/IgoryanJakhno/NeuroPharmAI)](https://github.com/IgoryanJakhno/NeuroPharmAI/releases)
[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

**Нейро-фарм** — это интеллектуальный помощник для поиска информации о лекарственных препаратах. Программа понимает запросы на естественном языке, находит аналоги, проверяет совместимость препаратов и анализирует побочные эффекты с помощью локальной нейросети Llama 3.1.

---

## 🚀 Возможности

- 🔍 **Поиск препаратов** по названию, действующему веществу, показаниям и симптомам
- 💊 **Поиск аналогов и синонимов** (включая терапевтические аналоги по фармакологической группе)
- ⚖️ **Сравнение двух препаратов** по ключевым атрибутам
- ⚠️ **Проверка совместного применения** двух препаратов на совместимость
- 📋 **Побочные эффекты** — детальный анализ на основе официальных инструкций (с поддержкой LLM)
- 🔎 **Умные фильтры** по производителю, стране, лекарственной форме и дозировке
- 🧠 **Интеграция с Llama 3.1** через Ollama для сложных запросов
- 🌐 **Парсинг инструкций** с авторитетных сайтов (rlsnet.ru)
- 👥 **Ролевая модель пользователей**: обычный пользователь, старший пользователь, разработчик
- 🔐 **Локальная аутентификация** с хешированием паролей и ограничением срока действия

## 📦 Установка

### Для обычных пользователей (рекомендуемый способ)

1. Перейдите в раздел **[Releases](https://github.com/goryanJakhno/NeuroPharmAI/releases)**.
2. Скачайте последнюю версию **`NeuroPharm_Setup.exe`**.
3. Запустите установщик и следуйте инструкциям на экране.
4. После установки запустите программу через ярлык на рабочем столе или в меню «Пуск».

> ✅ Установщик автоматически создаёт все необходимые папки и ярлыки. Никаких дополнительных действий не требуется.

## 👤 Первый запуск и управление пользователями

После установки программы при первом запуске вас встретит окно входа в систему.

### 🔑 Учётная запись по умолчанию

Для первоначальной настройки в системе создана учётная запись:

- **Логин:** `admin`
- **Пароль:** `Admin12345`

> ⚠️ **Это стандартный администратор, созданный для инициализации системы.**  
> Для обеспечения безопасности настоятельно рекомендуется сменить этот пароль или создать новых пользователей.

### 🔄 Создание новых пользователей

1. Войдите в систему под учётной записью `admin`.
2. В главном окне нажмите кнопку **👥 Пользователи** (доступна только для роли `dev`).
3. В открывшемся окне нажмите **➕ Добавить**.
4. Заполните поля:
   - **Логин:** только латинские буквы и цифры (минимум 3 символа)
   - **Пароль:** минимум 8 символов (латиница и цифры)
   - **Роль:** `user` (обычный), `senior` (старший) или `dev` (разработчик)
5. Нажмите **Создать**.

После создания новых пользователей рекомендуется **удалить учётную запись `admin`**, чтобы предотвратить несанкционированный доступ через стандартные учётные данные.

### 🗑️ Удаление учётной записи по умолчанию

1. Войдите под учётной записью **нового администратора** (с ролью `dev`).
2. Откройте окно **👥 Пользователи**.
3. Выберите пользователя `admin` в списке.
4. Нажмите **❌ Удалить** и подтвердите действие.

Теперь ваша система настроена и безопасна для использования.

### Для разработчиков (запуск из исходного кода)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/goryanJakhno/NeuroPharmAI.git
   cd NeuroPharmAI
   ```
Создайте виртуальное окружение и установите зависимости:

```bash
pip install -r requirements.txt
````
Запустите программу:

```bash
python gui_mgr.py
```
🧠 Настройка LLM (для работы с нейросетью)

Для использования функций анализа с ИИ (побочные эффекты, сравнение, проверка взаимодействия) необходимо установить Ollama и загрузить модель Llama 3.1.

Скачайте и установите Ollama с официального сайта.

Откройте терминал и выполните команду:

```bash
ollama pull llama3.1:8b
```
Убедитесь, что сервер Ollama запущен:

```bash
ollama serve
```
💡 Во время установки программы вы можете поставить галочку «Открыть страницу установки Ollama», чтобы сразу перейти на сайт.

🗂️ Структура проекта
```text
NeuroPharmAI/
├── gui_mgr.py              # Главный модуль графического интерфейса
├── data_base_mgr.py        # Ядро агента и работа с базой данных
├── query_parser.py         # NLU-парсер запросов (intent detection)
├── dbms_parser.py          # Формирование структурированных ответов
├── llm_mgr.py              # Менеджер LLM (Ollama)
├── site_parser.py          # Парсинг инструкций с сайтов
├── ftp_agent.py            # FTP-агент для обновления БД
├── auth.py                 # Аутентификация и управление пользователями
├── usr_mgr.py              # Менеджер пользователей
├── prompt_templates.json   # Шаблоны промптов для LLM
├── build.py                # Скрипт сборки .exe через PyInstaller
├── installer.iss           # Сценарий для Inno Setup
├── requirements.txt        # Зависимости Python
└── README.md               # Этот файл
```

2. Определение intent'ов в QueryParser

3. Хеширование паролей

4. Проверку срока действия пароля
